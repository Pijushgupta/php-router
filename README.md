#### Php Router

###### Clone this repository

```bash
git clone https://github.com/Pijushgupta/php-router

```

###### Adding it to the project

```php
include_once 'Location of file';
```

OR use Composer/auto-loader

###### Using it to the project

```php
use phpRouter\Router;
```

```php
new Router('/', function () {
	echo 'Home';
});
```
